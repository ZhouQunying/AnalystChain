## 🚀 快速开始

### 第一次使用

1. **阅读指南** → [任务管理指南.md](./任务管理指南.md)
2. **查看总览** → [主任务.md](./主任务.md)
3. **进入当前阶段** → [阶段1_知识基础.md](./阶段1_知识基础.md)
4. **开始执行任务** → 找到第一个 `- [ ]` 未完成任务

### 日常使用

```
早上：打开当前阶段文件 → 查看今天的任务
工作中：完成任务后 → 勾选checkbox ✅
晚上：更新进度条 → 写工作日志
```
